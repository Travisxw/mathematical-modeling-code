说明：
1. OLR.mp4为根据全球热辐射数据制作的视频，用于直观展示变化规律。
2. SST.mp4为根据全球海洋温度数据制作的视频，用于直观展示变化规律。
3. T1_1文件夹为用于读取加拿大各个城市的温度数据并进行数据处理及趋势函数拟合和相关图片的绘制。
4. T1 _2文件夹用于对加拿大数据进行季节性时间序列分析和预测。
5. T1_3文件夹用于读取全球海洋表面温度数据并进行可视化展示，运用db4小波对数据进行周期性分析。
6. T2文件夹分为数据预处理和GA-BP神经网络预测两部分，数据处理包含了对所有因变量的数据读取和保存，将读取的数据运用T1_2的时间序列预测模型进行因变量的预测，将所有数据放入PredictData.xlsx文件中，运行Genetic.m进行预测。
7. T3文件夹用于对数据进行MK突变检验和Person系数分析。
8. T4文件夹用于对共词矩阵进行聚类分析。

注：由于数据原始文件过大，附件中已做了删减，请在论文的链接网址进行数据获取。
